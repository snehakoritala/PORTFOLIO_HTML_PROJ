#  Sneha Portfolio Website

##  Description
This is a personal portfolio website built using HTML. It showcases my skills, projects, and contact information. The project focuses on using semantic HTML, accessibility best practices, and clean structure.

---

##  Features
- Semantic HTML structure (header, nav, section, article, footer)
- Navigation using anchor links
- Project showcase using articles
- Accessible form elements (labels, alt text)
- Clean and readable layout

---

##  Technologies Used
- HTML5
